# Technical Design PRD Template

> This file is the structural template referenced by the drafti-architect skill when generating a PRD.
> Sections can be reduced based on project scale.
> Small scale (1~2 days): §1, §2, §4, §6, §7 only / Medium scale and above: all sections

---

```markdown
# {Project Name} — PRD

> Created: {YYYY-MM-DD}
> Status: draft | review | approved
> Generated by: drafti-architect
> Estimated scale: Small | Medium | Large

---

### 1.1 Current State (As-Is)

[What state should be achieved after resolution.

### 1.3 Business/Technical Impact

[Why must this be solved now.
```
